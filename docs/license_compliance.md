## 3. 한국 공공저작물 검토

### 3.1 공공누리 유형 검토

한국 공공저작물은 공공누리(KOGL) 4유형 체계로 운영됩니다.

| 유형 | 출처표시 | 상업적 이용 | 변경 | 본 프로젝트 RAG 활용 가능성 |
|------|---------|----------|------|----------------------|
| 제1유형 | 필수 | 허용 | 허용 | ✓ 가능 |
| 제2유형 | 필수 | 금지 | 허용 | ✓ 가능 |
| 제3유형 | 필수 | 허용 | 금지 | △ 변경 금지로 인해 RAG 청크 분할 시 충돌 가능 |
| 제4유형 | 필수 | 금지 | 금지 | ✗ 변경 금지로 RAG 부적합 |

### 3.2 국립정신건강센터 자료 검토 결과

국립정신건강센터가 발간한 정신건강사례관리시스템(MHIS) 표준매뉴얼 등 주요 
자료가 모두 **공공누리 제4유형(출처표시 + 상업적 이용금지 + 변경금지)** 으로 
공개되어 있음을 확인하였습니다.

**결론**: 제4유형의 "변경금지" 조항이 RAG의 청크 분할·임베딩·발췌 과정과 
법적 충돌 가능성이 있어, **국내 공공저작물은 본 프로젝트에 사용하지 않기로 
결정**하였습니다.

## 4. WHO 자료 라이선스 상세 검증

### 4.1 CC BY-NC-SA 3.0 IGO 핵심 조건

- **BY (Attribution)**: 적절한 출처 표시
- **NC (NonCommercial)**: 비영리 목적으로만 사용
- **SA (ShareAlike)**: 각색물도 동일 라이선스로 공개
- **WHO 특별 조건**: WHO 보증 인상 금지, WHO 로고 사용 금지, 제3자 자료 별도 처리

### 4.2 CC 본부 공식 입장 (2025년 5월 발표)

Creative Commons 본부는 2025년 5월에 "Using CC-Licensed Works for AI Training" 
가이드를 발표하여 다음을 명확히 했습니다.

1. **AI 학습 일반 입장**: "AI 학습은 저작권법상 기본적으로 침해가 아님"
2. **RAG에 대한 명시적 언급**: RAG는 "출처 표시가 가능한 형태"로 CC 라이선스 
   정신에 가장 부합하는 AI 활용 방식
3. **NC 조건의 적용**: 비영리 목적 사용 시에는 모든 단계(임베딩·검색·응답 생성)가 
   비영리여야 함
4. **SA 조건 발동 시점**: 공개 공유 시에만 발동되며, 각색물은 동일 라이선스 
   필수

### 4.3 본 프로젝트의 충족 여부

| 조건 | 충족 방식 | 검증 |
|------|--------|------|
| BY | ATTRIBUTION.md, README, RAG 응답 모두에 출처 명시 | ✓ |
| NC | 광고·결제·상업적 데이터 활용 없음. 학술 포트폴리오 목적 | ✓ |
| SA | 본 프로젝트를 CC BY-NC-SA 4.0으로 공개 (3.0 IGO와 호환) | ✓ |
| WHO 보증 인상 금지 | README와 LICENSE에 명시적 면책 | ✓ |
| WHO 로고 금지 | 사용 안 함 | ✓ |
| 제3자 자료 별도 처리 | WHO 자료 내 외부 출처 표·그림 임베딩 제외 | ✓ |

## 5. 한국어 번역본의 법적 지위 검증

### 5.1 PFA 한국어판 검증

- **번역 주체**: 한국 월드비전 (World Vision Korea), 2013년
- **WHO IRIS 등재**: https://iris.who.int/bitstream/10665/44615/4/9789241548205_kor.pdf
  (정식 등재 확인)

### 5.2 번역물의 라이선스 상속

CC BY-NC-SA 라이선스에서 번역물은 **derivative work**에 해당하며, 
ShareAlike 조건에 따라 **원본과 동일한 라이선스를 상속**합니다.

한국 월드비전이 한국어판 번역을 수행했지만, WHO IRIS에 공식 등재되어 있으므로 
원본 CC BY-NC-SA 3.0 IGO 라이선스가 적용된다고 판단합니다.

**결론**: PFA 한국어판은 본 프로젝트에서 안전하게 활용 가능

### 5.3 mhGAP 한국어판 부재 확인

WHO mhGAP IG v2.0의 공식 언어 버전은 Arabic, French, Italian, Marathi, 
Russian, Spanish, Ukrainian으로, **한국어판은 존재하지 않습니다**.

**결론**: mhGAP는 영문 원본을 임베딩하고, Claude가 한국어로 번역하여 응답을 
생성하는 구조를 채택

## 6. 임베딩 기술 선택의 라이선스 함의

### 6.1 임베딩 모델 선택: BAAI/bge-m3

다국어(한·영) 통합 임베딩을 지원하는 BAAI/bge-m3 모델 채택:
- 라이선스: MIT License (상업적 이용 가능, 본 프로젝트와 무관하게 안전)
- 한국어와 영문을 같은 벡터 공간에서 처리 → 코드 단순화

### 6.2 Anthropic API의 데이터 처리

- **모델 학습 미사용**: Anthropic Commercial Terms에 의해 API 입력 데이터는 
  모델 학습에 사용되지 않음
- **보관 기간**: 2025년 9월 정책 업데이트로 30일에서 7일로 단축
- **본 프로젝트와의 양립성**: WHO 자료의 NC 조건과 충돌하지 않는다고 판단

### 6.3 회색 지대 인식

본 프로젝트가 임용 포트폴리오로 활용된다는 점에서 "비영리"의 엄격한 해석 
가능성을 인식합니다. 다만 다음 점에서 비영리로 판단됩니다.

- 직접적 금전 거래 없음
- 광고·결제 모델 없음
- 학술·교육 목적이 주된 사용 의도
- 본인의 영리 활동(유료 강의·컨설팅)에 사용하지 않음

CC 본부 가이드 문서들도 학술·교육 목적은 일반적으로 NC 범위 내로 해석합니다.
